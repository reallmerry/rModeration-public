# rModeration-public

A moderation addon for PlotSquared designed for servers with rank-based plot limits.

The addon integrates with LuckPerms and automatically manages plot ownership permissions based on player groups. It is intended for Creative and Plot servers where different ranks provide different plot limits and moderation capabilities.

## Features

* PlotSquared integration
* LuckPerms support
* Automatic rank-based plot moderation
* Flexible permission configuration
* Lightweight and easy to configure
* Designed for Minecraft 1.21.11
